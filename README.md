## Installation

### Using an AUR helper

To install Arturo, all you have to do is:
```bash
yay -S arturo
```
or:
```
paru -S arturo
```

### Manually

This package can be also installed manually:
```bash
git clone https://aur.archlinux.org/arturo.git
cd arturo
makepkg -si
```
